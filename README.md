# 个人博客系统：Admin

介绍：第一阶段开发完成，只实现了基本功能：

- 登录 + 后端权限验证 + Token
- 文章：CRUD
- 动态权限路由渲染 Sidebar
- 开关 Sidebar
- 拉取文章列表

## 使用技术

- Vue-CLI + Vue-Router + Vuex + ElementUI
- TUI-Editor + Less
- JS-Cookie
